1. Click the HSB tab at the ColorChooser. 2. Click in the lower left corner of the gradient palette in order to select a color such that all three RGB values are single digit colors (such as 0, 0, 0 or 5, 3, 1). 3. Click another tab, then click back to the HSB tab. 4. Now click the lighter colors that should have 2 and 3 digit RGB values (in the upper right corner). If all digits of each RGB value are shown then test passes. If only the last digit of their values are shown then test fails.
